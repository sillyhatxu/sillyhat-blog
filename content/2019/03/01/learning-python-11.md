# Two Dimensional List

[Link to my github](https://github.com/sillyhatxu/learning-python/blob/master/two_dimensional_list.py) (This is learning code.)

```
number_grid = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9],
    [0]
]

print(number_grid)

for row in number_grid:
    for col in row:
        print(col)
```
