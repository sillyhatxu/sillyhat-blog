#### Install
----
> Install hugo

`brew install hugo`

`hugo version`

> Create a New Site

`hugo new site sillyhat-hugo`

> Add a Theme

`cd sillyhat-hugo`

`cd themes`

`git clone https://github.com/giraffeacademy/ga-hugo-theme.git`

`echo 'theme = "ga-hugo-theme"' >> config.toml`

> Add Some Content

`hugo new A.md`
